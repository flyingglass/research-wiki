# 台积电 (TSMC / Taiwan Semiconductor Manufacturing Company)

- **成立**: 1987年
- **创始人/灵魂人物**: 张忠谋（Morris Chang）
- **总部**: 台湾新竹科学园区
- **核心业务**: 专业晶圆代工（Pure-Play Foundry）

## 商业模式：专业晶圆代工
台积电独创的商业模式：**只做晶圆制造，不做品牌/设计/终端产品**。

- 不与客户竞争（不做自品牌产品）
- 客户是伙伴（苹果、英伟达、高通、博通、AMD）
- 全球首屈一指的纯代工厂，定义Foundry行业

## 客户关系三原则
1. **不与客户竞争** — 只做代工，不碰IC设计，不推出自有产品
2. **不轻易承诺，一旦承诺不计代价完成** — 准时交付精确到天
3. **客户是伙伴** — "最快乐的事是看到客户成长赚钱"

## 竞争壁垒
- **客户信任**：数十年来积累，从1988年不计亏损完成英特尔订单开始
- **良率优势**：从日本/台湾的人力资源优势出发，持续积累
- **学习曲线**：大产能→成本降低→更多客户→更大产能的正循环
- **阿斯麦联盟**：全球首家采用ASML光刻机的半导体公司，与ASML共同成长
- **不干涉大股东**：开发基金6.4%的完美平衡

## 关键历史

| 年份 | 事件 |
|------|------|
| 1985 | 张忠谋任工研院院长，提出台积电方案 |
| 1987 | 台积电正式成立，政府48.3%+飞利浦27.6% |
| 1988 | 拿到英特尔MCU订单，17个月盈亏平衡 |
| 1990 | 二厂量产，开始用阿斯麦光刻机，袖珍洁净箱创新 |
| 1994 | 台湾上市（94.9.4 = "IC IC"） |
| 1997 | 美国ADR上市 |
| 2008 | 飞利浦完全退出（21年，135倍回报） |
