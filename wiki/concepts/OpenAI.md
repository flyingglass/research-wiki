# OpenAI

## 定义

2015 年由马斯克、Sam Altman、伊利亚·苏茨克维等人创立的人工智能公司。最初定位为非盈利的安全 AI 研究组织，后转型为盈利实体。推出 ChatGPT 引发 AI 产业爆发。

## 关键时间线

| 时间 | 事件 |
|------|------|
| 2015 夏 | 马斯克与拉里·佩奇因 AI 安全翻脸，组织硅谷瑰丽酒店聚餐 |
| 2015 底 | 伊利亚离开 Google 加入，OpenAI 成立 |
| 2016.8 | 英伟达将第一台 DGX-1 送 OpenAI（黄仁勋签名） |
| 2018 | 马斯克因特斯拉产能地狱退出 |
| 2019 | OpenAI 成立盈利组织，微软 10 亿→100 亿$投资 |
| 2020.6 | GPT-3 发布 |
| 2022.11.30 | ChatGPT 发布，黄仁勋称"AI 的 iPhone 时刻" |
| 2023 | 伊利亚逼宫 Sam Altman，失败后离开 |
| 2024.5 | 伊利亚创立 SSI（Safe Super Intelligence） |

## 与英伟达的关系

- 英伟达 DGX-1 第一个用户
- ChatGPT 引爆 GPU 需求，拯救 2022 年三重危机中的英伟达
- 伊利亚 → OpenAI → ChatGPT → 英伟达数据中心爆发：一条完整的因果链
